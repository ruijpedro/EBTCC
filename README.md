# Inspeções_RJP v1.1

App PWA para inspeções de manutenção de edifícios da Linha do Oeste, no troço Sabugo → Carriço.

## Novidades v1.1

- Fotografias nas inspeções
- Legendas nas fotografias
- Fotografias incluídas no PDF
- Numeração automática de inspeções
- Estados por estação: verde, amarelo e vermelho
- Histórico por estação/relatórios
- Botão **Gerar PDF**
