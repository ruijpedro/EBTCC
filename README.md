# Inspeções_RJP

App PWA para inspeções de manutenção de edifícios da Linha do Oeste, no troço Sabugo → Carriço.

## Inclui

- Dashboard
- Lista de estações
- Nova inspeção
- Checklist C / NC / NA
- Observações
- Guardar inspeções em localStorage
- Botão **Gerar PDF**
- Impressão/exportação em PDF com layout semelhante à ficha de inspeção

## Estações

A lista remove **Alcaínça-Moinhos**, por ter sido demolido.

## Como usar no GitHub

1. Criar repositório `inspecoes-rjp`
2. Enviar todos os ficheiros deste ZIP
3. Ativar GitHub Pages:
   - Settings
   - Pages
   - Deploy from branch
   - Branch `main`
   - Folder `/root`
